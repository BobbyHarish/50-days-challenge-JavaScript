Array Iterations in JavaScript-Part 1

Array forEach() :

forEach() function is used to Assign value like space,text or other value for each Array Elements.

Let us consider an array that contains number values.

Array = [8,7,4,13,74,88,44]

let us display this array Elements one after other vertically.

Code :

        const Array=[8,7,4,13,74,88,44];
        let ArrayElements="";
        Array.forEach(myfunction);
        document.getElementById("demo").innerHTML=ArrayElements;

        function myfunction(value)
        {
            ArrayElements+=value+"
"
        }

Array Map() :

Syntax:array_name.map(function_name)

Let us return the above array with multiple of 2.

Code :

        const Array2=Array.map(myfunction2)
        document.getElementById("demo1").innerHTML=Array2;

        function myfunction2(value,index,array)
        {
          return value*2 ;
        }

Array flatMap() :

Syntax:array_name.faltMap(Expression)

Let us return the above array with multiple of 3 using flatMap

Code :

        const Array3=Array.flatMap((x) => x*3);
        document.getElementById("demo2").innerHTML=Array3;

Array filter() :

Syntax:array_name.filter(function_name)

filter() are used to display specific Elements

let us display the specific Elements of this array.

First we display Elements over 4.

Code :

        const Array4=Array.filter(myfunction3)
        document.getElementById("demo3").innerHTML=Array4;

        function myfunction3(value,index,array)
        {
            return value > 4
        }

Now we display Elements Under 74.

Code :

        const Array5=Array.filter(myfunction3_1)
        document.getElementById("demo4").innerHTML=Array5;

        function myfunction3_1(value,index,array)
        {
            return value < 74
        }

Array reduce() :

Syntax:array_name.reduce(function_name)

reduce() is used to dispaly sum , sub of total Elementsin array.

let us dispaly sum of total Elements in the array

Code :

        let sum=Array.reduce(myfunction4)
        document.getElementById("demo5").innerHTML=sum;

        function myfunction4(total,value,index,array)
        {
            return total+value;
        }
